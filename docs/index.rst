Vector
======

.. epigraph::

   A Minecraft proxy. Fast. Open. Free. Always.

   A spiritual successor to Velocity built on Kotlin + Netty.

Vector is a clean-room reimplementation of Velocity's proxy functionality with a
better architecture, native coroutines, a real terminal, cluster-aware design, and
an AGPL-3.0 license.
